# Web-starter

A simple web starter project

## Demo

[https://web.loke.dev](https://web.loke.dev)

## Getting Started

These instructions will get you a copy of the project up and running on your local machine for development and testing purposes. See deployment for notes on how to deploy a copy of the project on Netlify.

### Prerequisites

- Node
- Yarn

### Development

A step by step series of examples that tell you how to get a development env running

Clone the repo

```
git clone git@github.com:lokecarlsson/web-starter.git
```

Install dependencies

```
yarn
```

Run the dev server

```
yarn dev
```
